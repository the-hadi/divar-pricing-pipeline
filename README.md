# AI Quera Bootcamp – Project 1

This repository contains the solution for the **First Project of the AI Quera Bootcamp (Spring 2026)**.

## Project Overview

The project is based on a real-world **Divar real estate listings** dataset containing over one million property advertisements across Iran.

The project consists of three main parts:

- **Descriptive Statistics:** Exploratory data analysis and visualization of the housing market.
- **Hypothesis Testing:** Statistical analysis of common real-estate assumptions.
- **Machine Learning:**
  - Property clustering using **K-Means** and **DBSCAN** for recommendation systems.
  - Property price prediction using supervised machine learning models.

## Repository Structure

- `descriptive_statistics/` – Exploratory data analysis and visualizations.
- `hypothesis_testing/` – Statistical hypothesis testing.
- `recommender_system/` – Clustering models.
- `prediction_model/`   – Classification models.
